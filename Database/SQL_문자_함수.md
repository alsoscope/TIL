- CONCAT<br>
CONCAT('Hello', 'world') → Helloworld<br>
문자 결합. 파라미터는 2개만 사용 할 수 있다.

- SUBSTR<br><br>
SUBSTR('HelloWorld', 1, 5) → Hello<br>
지정한 길이의 문자열 출력

- LPAD<br>
LPAD(salary, 10, '*') → *****24000<br>
문자 값을 오른쪽 정렬하고 빈곳을 지정한 문자열로 채운다.
